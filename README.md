# AStarFindingPath
#### 使用UGUI完成的A星寻路demo。
#### 寻路用的是曼哈顿估价法。
#### 原理介绍：[莫水千流的猫猫寻路](https://www.cnblogs.com/zhoug2020/p/3468167.html)
#### 原理可视化：[链接](https://anseyuyin.github.io/AStar-Process/demo/)
#### Unity版本：2017.2.1f1 (64-bit)
#### 使用方法：鼠标左键设立起点，鼠标右键设立终点即可。
